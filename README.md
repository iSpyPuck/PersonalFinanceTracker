# Personal Finance Tracker

Track income, expenses, and budgets with interactive charts.

## Tech

- React  
- Recharts  
- Client side data management

## Overview

The Personal Finance Tracker is a financial management tool that helps users take control of their money. Track every dollar with detailed categorization, set budgets, and see insights into spending patterns.

The dashboard features interactive charts and graphs that visualize financial data over time, which makes it easy to spot trends and find areas to improve. Features include recurring transaction flags and multi account support.

This project highlights data visualization, financial calculations, chart libraries, state management, and building tools that are useful in everyday life.
